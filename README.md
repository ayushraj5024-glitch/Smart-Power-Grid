# ⚡ Wind Power Prediction System

A machine learning + physics-based smart wind power prediction system built using:

- Python
- Flask
- Machine Learning
- HTML/CSS/JavaScript
- XGBoost
- Random Forest
- Stacking Regressor

The project predicts wind turbine power generation using real-time weather parameters such as:

- Temperature
- Humidity
- Wind Speed
- Cloud Cover
- Atmospheric Pressure
- Rain Conditions

✅ Interactive Frontend UI
